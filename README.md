# CNN-DeepLearning-project
Built an Image Classification model using PyTorch and CNNs to classify food images such as Pizza, Steak, and Sushi. The project includes data preprocessing, augmentation, custom dataset creation, model training, evaluation, and GPU acceleration using CUDA. Strengthened practical skills in Deep Learning, Computer Vision, and the PyTorch workflow.
Data zipfile is in repository but you can directly download data using code in a project file
